# 🎬 CineNexus

##📖 Description
A web application that allows users to search for movies, view details, and manage a personal watchlist using data fetched from the OMDb API.
___

## 🎯 Features
- 🔍 Search movies by title  
- 🎬 View movie details (poster, year, rating, genre, etc.)  
- ⭐ Add or remove movies from watchlist  
- ❤️ Mark movies as favourites
- 🎯 Advanced filtering by genre  
- ↕️ Sort movies by rating or year  
- 📱 Responsive user interface  
___

## 🌐 API Used
[TMDb API](https://www.themoviedb.org/)
> Note: An API key is required to use the TMDb API.
___

##🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Fetch API
___

##🚀 How to Run
- Clone the repository
- Open index.html in browser
